# 🤖 Telegram Bot

Simple Telegram bot using `python-telegram-bot` library.
Reference: https://python-telegram-bot.org/

First of all, you need to create your telegram bot first
https://t.me/botfather

Then save your bot token for API

---

# 📦 1. Clone Project use your terminal

```bash
git clone https://github.com/hakimcodez/telegram_bot.git
```

# 2. Open Project Folder
```bash
cd telegram_bot
```

# 3. Create a virtual environment
```bash
python -m venv venv
```
# 4. Run a virtual environment
### If you are using windows cmd
```bash
.\venv\Scripts\acticvate.bat
```

### If you are using unix/linux distro
```bash
source ./venv/Scripts/acticvate
```

# 5. Now install required libraries into your virtual environment.
```bash
pip install -r requirements.txt
```

# 6. The most important step
- Create new file named `.env` not ~~.env.txt~~ then Put your bot token inside it
Inside **.env** file, use this format.
```bash
TOKEN=paste your bot token here
```

# 7. Run your own bot
```bash
python main.py
```

# Congrats, your telegram bot made with python is already running.
